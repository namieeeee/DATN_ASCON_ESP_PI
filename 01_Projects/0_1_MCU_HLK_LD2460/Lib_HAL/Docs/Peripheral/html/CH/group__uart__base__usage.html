Base Usage
UART Interface » Usage Flow

Description

在基础用法中, UART 有三种传输模式: Blocking ，Interrupt 和 DMA

Functions

UART0~5

特点 功能 说明 注意事项
Blocking 模式 Transmit 通过 HAL_UART_Transmit API
在 Blocking 模式下发送数据 		超时设置不应太短
Receive 通过 HAL_UART_Receive API
在 Blocking 模式下接收数据 		超时设置不应太短
IT 模式 Transmit 通过 HAL_UART_Transmit_IT API
在 Interrupt 模式下异步发送数据 		传输完成的时候，会产生一个中断
Receive 通过 HAL_UART_Receive_IT API
在 Interrupt 模式下异步接收数据 		传输完成的时候，会产生一个中断
DMA 模式 Transmit 通过 HAL_UART_Transmit_DMA API
在 DMA 模式下发送数据 		传输完成的时候，会产生一个中断
Receive 通过 HAL_UART_Receive_DMA API
在 DMA 模式下接收数据 		传输完成的时候，会产生一个中断
Pause 通过 HAL_UART_DMAPause API
暂停 DMA 传输 		使用 HAL_UART_DMAResume 恢复
Resume 通过 HAL_UART_DMAResume API
恢复 DMA 传输 		传输会继续直至传输完成
Stop 通过 HAL_UART_DMAStop API
停止 DMA 传输 		停止以后，传输不能恢复
