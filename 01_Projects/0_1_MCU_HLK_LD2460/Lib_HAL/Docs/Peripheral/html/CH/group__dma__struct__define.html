uint32_t ch_priority

Specifies the software priority for the DMA stream. This parameter can be a value of DMA Channel Priority

uint32_t ch_fifo_th

Specifies the FIFO threshold level. This parameter can be a value of DMA FIFO Threshold Level

uint32_t ch_sync

Specifies the synchronization logic enable for channel. When DMA PCLK(AHB bus) is different from PCLK(APBn bus), current filed need be enable, others keep disable. This parameter can be a value of HAL_ENABLE or HAL_DISABLE
