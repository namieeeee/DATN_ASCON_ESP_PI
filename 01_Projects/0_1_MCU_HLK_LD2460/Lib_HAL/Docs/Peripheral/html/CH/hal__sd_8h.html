◆ __HAL_SD_ENABLE_IT

#define __HAL_SD_ENABLE_IT (   __HANDLE__,
  __INTERRUPT__ 
)    __SDMMC_ENABLE_IT((__HANDLE__)->Instance, (__INTERRUPT__))

Enable the SD device interrupt.

Parameters
<strong>HANDLE</strong>SD Handle
<strong>INTERRUPT</strong>specifies the SDMMC interrupt sources to be enabled. This parameter can be one or a combination of the following values:
  • SDMMC_IT_CMDCPLIE: Command complete interrupt enable
  • SDMMC_IT_TRANSCPLIE: Transfer complete interrupt enable
  • SDMMC_IT_BLKGAPIE: Block gap event interrupt enable
  • SDMMC_IT_DMAIE: DMA interrupt enable
  • SDMMC_IT_BUFWIE: Buffer write ready interrupt enable
  • SDMMC_IT_BUFRIE: Buffer raed ready interrupt enable
  • SDMMC_IT_CARDINSTIE: Card insertion interrupt enable
  • SDMMC_IT_CARDRMIE: Card remove interrupt enable
  • SDMMC_IT_CARDIE: Card interrupt enable
  • SDMMC_IT_CMDTOEIE: Command timeout error interrupt enable
  • SDMMC_IT_CMDCRCEIE: Command CRC error interrupt enable
  • SDMMC_IT_CMDEBEIE: Command end bit error interrupt enable
  • SDMMC_IT_CMDINDEIE: Command index error interrupt enable
  • SDMMC_IT_DATTOEIE: Data timeout error interrupt enable
  • SDMMC_IT_DATCRCEIE: Data CRC error interrupt enable
  • SDMMC_IT_DATEBEIE: Data end bit error interrupt enable
  • SDMMC_IT_CURLMTEIE: Current limit error interrupt enable
  • SDMMC_IT_ATCMDEIE: Auto CMD12 error interrupt enable
  • SDMMC_IT_ADMAEIE: ADMA error interrupt enable
Return values
NoneEnable the SD device interrupt
