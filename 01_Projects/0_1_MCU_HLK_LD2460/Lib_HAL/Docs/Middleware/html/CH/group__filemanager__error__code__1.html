fatfs error code
FileManager » Config Parameter

FR_DISK_ERR

磁盘I/O层硬错误, 例如没有介质

FR_INT_ERR

断言错误, 可能原因如下：

  • 卷上的FAT结构有错误
  • FatFs模块自身存在bug
  • 底层实现错误

FR_NOT_READY

磁盘未准备好, 可能原因如下：

  • 驱动器中无介质
  • 底层实现错误
  • 硬件配置错误
  • 存储设备已损坏

FR_NO_FILE

当前目录下找不到文件

FR_NO_PATH

找不到路径中的目录

FR_INVALID_NAME

路径名格式无效, 可能原因如下：

  • 文件名中存在不允许的字符
  • 文件名超出8.3格式
  • 字符串内存在编码错误
  • FF_MAX_LFN小于当前文件名长度。（长文件名情况下）

FR_DENIED

拒绝访问, 可能原因如下：

  • 使用写模式打开只读文件
  • 删除只读文件或目录
  • 删除非空目录或当前目录
  • 读取未使用FA_READ标志打开的文件
  • 对未使用FA_WRITE标志打开的文件进行任何修改
  • 由于根目录已满或磁盘已满，无法创建对象
  • 无法为文件分配连续区域

FR_NO_FILE

当前目录下找不到文件

FR_EXIST

目录中已经存在具有相同名称的对象

FR_INVALID_OBJECT

对象无效或损坏

FR_WRITE_PROTECTED

磁盘已写保护

FR_INVALID_DRIVE

无效的驱动器号

FR_NOT_ENABLED

驱动器未启用

FR_NO_FILESYSTEM

磁盘中没有文件系统

FR_MKFS_ABORTED

文件系统格式化被取消

FR_TIMEOUT

磁盘操作超时

FR_LOCKED

文件被锁定

FR_NOT_ENOUGH_CORE

内存不足

  • 无法为 LFN 工作缓冲区分配内存（相关选项：FF_USE_LFN）
  • 给定缓冲区的大小不足以满足所需的大小。

FR_TOO_MANY_OPEN_FILES

文件打开太多

FR_INVALID_PARAMETER

参数无效
