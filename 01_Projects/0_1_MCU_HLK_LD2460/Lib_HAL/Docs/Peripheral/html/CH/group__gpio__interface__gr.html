Description

GPIO (General-purpose input/output) input/output data from systems and devices. Each GPIO can be set as input or output.
Rising-Edge, Falling-Edge, Rising and Falling Edge, High/Low level can be set to trigger GPIO interrupt.


Feature

  1. Up to 32 independent input, output, and output enable buses for bidirectional I/O pins
  2. Each port can be bypassed.
  3. Each port can individually trigger GPIO interrupt when programmed as input pin.
  4. Each port of interrupt generation can be triggered at rising edge, falling edge, both edges, or at high/low level.
  5. Each port can be pulled high or pulled low.
  6. Each port can choose pre-scaled or PCLK clock source.
  7. Output data bit can be separately set or cleared.
  8. All ports are set as inputs upon hardware reset.
  9. If the clock is turned off, rising and falling edges can be detect. Rising and falling interrupt status of each port can be independently masked
