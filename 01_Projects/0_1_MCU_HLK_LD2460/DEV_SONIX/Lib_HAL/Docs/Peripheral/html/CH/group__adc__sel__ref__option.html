ADC Interface » Usage Flow

Description

ADC有三种电压参考源选择(电压参考源作为输入电压转换值的参考，当输入电压大于等于参考源电压时，其转换值为 0xFFF )：

  • 内置电压参考源 VDD：使用板子的运行电压 VDD 作为参考电压源
  • 内置电压标准参考源： ADC 额外提供了三种标准电压作为参考源，当使用这种参考源时，需要给 AIN0 接一个 1uF 的电容，此时不能输入电压到 AIN0 进行转换，三种标准电压：
    • 1.5 V
    • 2.0 V
    • 2.5 V
  • 外置电压参考源：输入电压到 AIN0 作为电压参考源
