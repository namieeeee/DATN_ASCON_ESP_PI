Description

使用Flash驱动控制和访问 Flash。驱动在非 OS 上运行。

Flash 内存由32位宽的存储单元(cell)构成，使用这些单元可以存储代码与数据常量，Flash 内存位于芯片内存映射的某个 Base 地址。

芯片高性能的 Flash 内存具有下面的主要特点：

扇区 说明
User ROM 至多504K，分成了512字节的1008分页
Boot ROM 至多8K，分成了512字节的16分页
Information Block 至多512字节的4分页


Flash 内存组织


Features

  1. The MCU is stalled during Flash program and erase operations, although peripherals (Timers, WDT, I/O, PWM, etc.) remain active.
  2. Watchdog timer should be cleared if enabled before the Flash write or erase operation.
  3. The erase operation sets all the bits in the Flash page to logic 0.
  4. HW will hold system clock and automatically move out data from RAM and do programming, after programming finished, HW will release system clock and let MCU execute the next instruction.
