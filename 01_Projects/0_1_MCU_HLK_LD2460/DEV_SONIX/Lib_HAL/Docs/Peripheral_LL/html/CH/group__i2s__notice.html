Notice
I2S Interface
  1. 在SN34F28x中，I2S和SPI共用同一个外设，即同一时间内只能使用I2S或SPI两者之一
  2. 为确保传输正确，请设定较高的SYSCLK和APB1CLK，如48M以上
    • SYSCLK：系统时钟
    • APB1CLK：APB1总线的时钟，由系统时钟经分频系数进行分频得到，分频系数推荐采用 LL_RCC_HCLK_DIV2

  3. 为确保传输正确，(SSPCLK / APB1CLK) 的数值范围应在[1,4]，推荐为2
  4. Master 和 Slave 通信时, 请先调用 Slave 的接口, 再调用 Master 的接口;
