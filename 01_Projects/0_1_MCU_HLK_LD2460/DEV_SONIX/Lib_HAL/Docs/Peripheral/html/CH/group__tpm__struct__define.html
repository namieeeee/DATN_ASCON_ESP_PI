◆ TPM_Init_T

struct TPM_Init_T
Data Fields
uint32_t pre_scaler

Specifies the Pre-Scaler value used to divide the HAL TPM clock. This parameter can be a number between Min_Data = 0x00 and Max_Data = 0xFF.

Note
when PC(Pre-scale Counter) equal PreScaler, TC(TPMer Counter) will increment or reduce.
uint32_t mode

Specifies the mode that decide when to increment or reduce TC or PC. default valve is TPM_CTM_TIMER. This parameter can be a value of TPM Mode .

Note
in TPMer mode, PC will increment or reduce at every rising PCLK edge.
in counter mode, PC will increment or reduce at CAP pin rising(or falling, or both) edge.
in counter mode, can not use capture function.
in counter mode, the duration of the HIGH/LOW levels on the same CAP input cannot be shorter than 1 / (2 * PCLK).

◆ TPM_OC_T

struct TPM_OC_T
Data Fields
uint32_t mode

Specifies the TPM channle mode. This parameter can be a value of TPM Output Compare and PWM Modes

uint32_t period

Specifies the Period of current channel. This parameter can be a number between Min_Data = 0x0000 and Max_Data = 0xFFFF.

uint32_t stop

Specifies the enable of current channel to stop counter when TC equal Match Register. This parameter can be a value of HAL_ENABLE of HAL_DISABLE .

uint32_t reset

Specifies the enable of current channel to reset counter when TC equal Match Register. This parameter can be a value of HAL_ENABLE of HAL_DISABLE .

uint32_t idle_state

Specifies the TPM output compare pin state during idle state in EMC mode. This parameter can be a value of TPM output compare pin state during idle state in EMC mode.

Note
This parameter is valid only for EMC mode.

◆ TPM_PWMN_T

struct TPM_PWMN_T
Data Fields
uint32_t mode

Specifies PWMN mode. This parameter can be a value of TPM PWMN Modes

uint32_t db

Specifies Count of PWMN output dead-band period time. This parameter can be a number between Min_Data = 0x0000 and Max_Data = 0x3FF. PWMN output dead-band period time=db * CT16Bn_PCLK*(PR+1) cycle. Recommend the dead-band period less than PWM high pulse width, or the PWM high pulse width disappears
