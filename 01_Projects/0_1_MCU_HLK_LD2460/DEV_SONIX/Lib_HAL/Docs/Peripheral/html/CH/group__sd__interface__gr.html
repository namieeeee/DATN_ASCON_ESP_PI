Description

使用SD 访问与控制 SD 内存卡。


特点

  1. 符合 SD Host 控制器标准规范2.0版
  2. 支持 DMA 以及非 DMA 数据传输
  3. 符合 SD 物理层规格2.0版
  4. 支持可配置的 SD 总线模式：4位模式和8位模式
  5. 符合 SDIO 卡规格2.0版
  6. 符合 MMC 卡规格，版本4.2强制性部分
  7. 对数据 FIFO 支持可配置的 1K SRAM
  8. 支持可配置的1位/4位 SD 卡总线和1位/4位/8位 eMMC 卡总线
  9. 对7位和16位 CRC 数据内置的产出与检查
  10. 支持基于 SDIO 读写机制


配置设置 (sn34f78x_hal_conf.h)

  1. 需要打开 configUSE_SD_PERIPHERAL
  2. 如果要使用 Block 模式，需要打开 configUSE_SD_BLOCKING
  3. 如果要使用 Interrupt 模式，需要打开 configUSE_SD_NONBLOCKING_IT
  4. 如果要使用 DMA 模式， 需要打开 configUSE_SD_NONBLOCKING_DMA
  5. 如果要使用回调函数，需要打开 configUSE_SD_REGISTER_CALLBACKS


信号线

信号 名称 说明
CMD SD Command SD 命令发送以及响应接收
CLK SD Clock 正常速度(25MHz) 或者高速度 (50Mhz)
DATA 0-7 SD DATA0 - DATA7 发送数据
