Description

NORFlash 调试模块，对Flash访问的功能模块化，使用简单，便于调试；目前支持对 MX25L6433F 闪存的访问，访问协议为spi；
提供了例如Program/Read/Protect等多种API，可直接调用实现相应功能。


  • 特点
  1. 提供Program/Read/Erase等适用所有闪存类型的通用接口
  2. 支持通过spi协议对 MX25L6433F 闪存进行访问，并提供针对该闪存的API接口
  3. 可扩展对其他类型闪存的访问


  • 配置设置 (sn34f78x_hal_conf.h)
  1. 需要 configUSE_NORFLASH_MIDDLEWARE = 1
  2. 需要先在 sn34f78x_hal_conf.h 中选定 Flash 类型
  3. 如果要使用 Blocking 接口，需要设定 configUSE_NORFLASH_BLOCKING = 1
  4. 如果要使用 DMA 非阻塞接口，需要设定 configUSE_NORFLASH_NONBLOCKING_DMA = 1，且需要芯片支持
  5. 如果要使用 注册函数 接口， 需要设定 configUSE_NORFLASH_REGISTER_CALLBACKS = 1


  • 功能
    NORFlash 支持下面的功能：
    功能 说明 注意事项
    写/读/擦除 Flash 实现对Flash的基本操作 a. 写/擦除时，需要注意操作的区域不是处于写保护状态
    b. 当处于Deep power down 模式时，除了退出DPD的操作，Flash将禁止一切访问
    保护指定区域机制 保护指定的区域后，该区域将禁止所有的写入与擦除，读操作不受影响 提供了退出保护的API
    Deep Power Down 模式 进入DPD模式后，除了退出DPD的操作，Flash将禁止一切访问 提供了退出DPD的API


  • 注意事项
  1. 目前支持SPI Flash的板子有 Evaluation board，该板子已经硬件固定了 Flash 外设，因此只能使用SPI0，且 DMA 模式只支持 DMA1
  2. Program 指定区域之前，需要先 Erase，否则可能写入错误

NORFlash protocol

NORFlash 可扩展不同Flash类型，也可扩展不同通信协议与Flash进行交互；目前支持Flash与协议如下：

Flash类型 支持协议 说明
MX25L6433F 闪存 SPI 需要 configUSE_NORFLASH_TYPE = NORFLASH_MX25L6433F
