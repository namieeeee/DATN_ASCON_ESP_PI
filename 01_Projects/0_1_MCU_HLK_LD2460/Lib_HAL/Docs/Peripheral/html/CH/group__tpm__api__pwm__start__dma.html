Description

开启 Channel 输出 PWM，且开启 DMA 传输 Memory 数据修改 MR 值

[in]data指向数据缓冲
[in]length数据缓冲的长度
Returns
Execution Status
  1. 使用该函数前必须确保 channel 已通过 HAL_TPM_PWM_ConfigChannel 设置为 PWM channel
  2. 使用该函数前必须确保 channel 状态为 Ready
  3. 使用该函数后 TPM 以及对应的 channel 状态为 Busy
  4. 使用该函数后相应的Pin(AFIO设置)会根据 HAL_TPM_PWM_ConfigChannel 设置输出相应信号
  5. 当 channel 的 MR 等于 TC 且满足 TPM Counting Mode 中的域 "cm" 时通过 DMA 传输内存数据修改 MR 值，后根据 HAL_TPM_PWM_ConfigLoadMode 设定在特殊时刻才会实际生效，若不支援加载模式的 Channel 会在 TC == 0 时生效
  6. DMA 传输完成之后默认情况下调用 HAL_TPM_PWM_PulseFinishedCallback
  7. 使用该函数必须在 sn34f78x_hal_conf.h 中将 configUSE_TPM_PWM 以及 configUSE_TPM_DMA 设置为1
