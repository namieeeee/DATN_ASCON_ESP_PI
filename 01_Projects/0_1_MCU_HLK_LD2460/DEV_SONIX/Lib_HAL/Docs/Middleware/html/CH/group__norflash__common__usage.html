NORFlash 模块文件说明：

项目 内容 说明
头文件 Middleware_norflash.h 用于导入 NORFlash 模块，且描述模块实现的宏与结构体
NORFlash_MX25L6433F.h 描述对特定Flash MX25L6433F 实现的宏与结构体
源文件 NORFlash.c NORFlash 模块 API 的实现文件，包含所有Flash类型的接口
MX25L6433F.c MX25L6433F Flash自己的函数接口



当前 NORFlash 模块所支持的功能：

类别 所属 说明 注意事项
通用接口 所有Flash适用 通过 HAL_NORFlash_Program API
将数据写入Flash 		a. 若写失败，判断Flash是否处于写保护状态或DPD模式;
b. 需要 configUSE_NORFLASH_BLOCKING = 1
通过 HAL_NORFlash_Read API
从Flash中读出数据 		a. 若读失败，判断Flash是否处于DPD模式;
b. 需要 configUSE_NORFLASH_BLOCKING = 1
通过 HAL_NORFlash_EraseSector
擦除Flash指定扇区 		a. 若失败，判断Flash是否处于写保护状态或DPD模式;
b. 需要 configUSE_NORFLASH_BLOCKING = 1
通过 HAL_NORFlash_EraseChip
擦除整块Flash 		a. 若失败，判断Flash是否处于写保护状态或DPD模式;
b. 需要 configUSE_NORFLASH_BLOCKING = 1
通过 HAL_NORFlash_ReadID
返回Flash ID 		a. 不同Flash返回的ID长度可能不同
b. 需要 configUSE_NORFLASH_BLOCKING = 1
特有接口1 MX25L6433F
Flash 适用 		HAL_NORFlash_Program_DMA API
HAL_NORFlash_Read_DMA API 		a. 提供了写/读的DMA接口，注意写保护或DPD模式
b. 需要 configUSE_NORFLASH_NONBLOCKING_DMA = 1
HAL_NORFlash_EraseBlock32
HAL_NORFlash_EraseBlock64 API 		a. 提供了以32KB或64KB大小擦除Flash
b. 需要 configUSE_NORFLASH_BLOCKING = 1
HAL_NORFlash_ProtectBlock API
保护指定区域 		a. 该区域将变成保护状态，且无法写入
b. 需要 configUSE_NORFLASH_BLOCKING = 1
HAL_NORFlash_ProtectBlockAll API
保护整块区域 		a. 所有区域将变成保护状态，且无法写入
b. 需要 configUSE_NORFLASH_BLOCKING = 1
HAL_NORFlash_ProtectResumeAll API
整块区域退出保护状态 		a. 回到 standby 状态，可正常写入数据
b. 需要 configUSE_NORFLASH_BLOCKING = 1
HAL_NORFlash_EnterDPD API
Flash进入Deep power down 模式 		a. 进入DPD后，Flash只接受 HAL_NORFlash_ExitDPD
b. 需要 configUSE_NORFLASH_BLOCKING = 1
HAL_NORFlash_ExitDPD API
Flash退出Deep power down 模式 		a. 回到 standby 状态
b. 需要 configUSE_NORFLASH_BLOCKING = 1
