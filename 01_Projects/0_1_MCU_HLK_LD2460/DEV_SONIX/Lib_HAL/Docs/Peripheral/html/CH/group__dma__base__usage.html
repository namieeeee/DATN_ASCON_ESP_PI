Blocking 模式 Start 使用 HAL_DMA_Start API
开启DMA在 Blocking 模式下传输数据
Abort 使用 HAL_DMA_Abort API
DMA在 Blocking 模式下暂停数据传输
Resume 使用 HAL_DMA_Resume API
使DMA从中止状态恢复，在 Blocking 模式下继续传输数据
Interrupt 模式 Start 使用 HAL_DMA_Start_IT API
开启DMA通过中断异步传输数据 		传输完成的时候，会产生一个中断
Abort 使用 HAL_DMA_Abort_IT API
DMA暂停通过中断异步传输数据 		暂停完成的时候，会产生一个中断
Resume 使用 HAL_DMA_Resume_IT API
恢复之前中止的传输
