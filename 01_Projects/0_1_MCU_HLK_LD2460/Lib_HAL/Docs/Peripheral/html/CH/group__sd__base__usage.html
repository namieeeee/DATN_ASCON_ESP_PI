Base Usage
SD Interface » Usage Flow

Description

SD 有三种传输模式: Blocking ， Interrupt 和 DMA。

Functions

模式 功能 说明 注意事项
Blocking 模式 写 通过 HAL_SD_WriteBlocks API
在 Blocking 模式发送一个或者数个 Block 的数据 		超时设置不应太短
读 By HAL_SD_ReadBlocks API
在 Blocking 模式接收一个或者数个 Block 的数据 		超时设置不应太短
Interrupt 模式 写 通过 HAL_SD_WriteBlocks_IT API
通过 Interrupt 异步发送一个或者数个Block的数据 		发送完成的时候，产生中断
读 通过 HAL_SD_ReadBlocks_IT API
通过 Interrupt 接收发送一个或者数个Block的数据 		发送完成的时候，产生中断
DMA 模式 写 通过 HAL_SD_WriteBlocks_SDMA API
SD 使用 DMA 模式发送数据。 		发送完成的时候，产生 DMA 中断
读 通过 HAL_SD_ReadBlocks_SDMA API
SD 使用 DMA 模式接收数据。 		发送完成的时候，产生 DMA 中断
