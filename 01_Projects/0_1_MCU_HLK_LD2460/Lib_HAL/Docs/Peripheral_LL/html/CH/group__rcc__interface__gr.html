Description

RCC (复位与时钟控制) 驱动提供两类函数：

  1. 复位函数包含了 Reboot 命令，外设 Rest 命令
  2. 时钟函数包含了 Oscillator 控制，时钟源配置以及外设时钟开启/关闭


注意事项

关联的驱动器

  • 使用 SCU 驱动通知 RCC 驱动需要的事件
  • 开启 HCLK 的时候，会使用 Flash 驱动。根据 HCLK 频率更新 Flash 等待状态

IRQ处理程序

  • 由 SCU 驱动提供的 LL_SCU_IRQHandler 函数需在 IRQ 函数 SCU_IRQHandler() 执行
  • 由 RCC 驱动提供的 LL_LVD_IRQHandler 函数需在 IRQ 函数 LVD_IRQHandler() 执行。如果使用了 LVD 函数
  • 上面两个函数通常是在C文件 sn34fxxx_it.c 中定义的，这个文件是由用户编程的


配置设置

在 sn34f78x_ll_conf.h 中：

  1. 如果使用 EHS，配置 LL_EHS_FREQ_MHZ 和 LL_EHS_INR_DS
