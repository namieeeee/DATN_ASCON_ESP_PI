PLL entry clock source. This parameter must be a value of PLL Clock Source Selection

uint32_t NS

programmable loop divider. PLLCLK = FREF * NS /FS This parameter can be a number between Min_Data = 6 and Max_Data = 80

uint32_t FS

PLL frequency range output divider. This parameter must be a value of PLL Clock Divider
