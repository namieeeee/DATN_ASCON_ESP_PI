Description

LCM (LCD模块) 控制功能与8080接口兼容，可与显示屏通信， 支持8位，9位，16位，18位类型的LCD显示屏模块。


  • 特点
  1. Support 8-bit, 9-bit, 16-bit and 18-bit data bus panels
  2. Support 8080 modes
  3. Support chip select pin
  4. LCM write data FIFO function


  • 配置设置 (sn34f78x_hal_conf.h)
  1. 需要打开configUSE_LCM_PERIPHERAL
  2. 如果要使用阻塞模式， 需要打开configUSE_LCM_BLOCKING
  3. 如果要使用DMA模式， 需要打开configUSE_LCM_NONBLOCKING_DMA
  4. 如果要使用回调，需要打开configUSE_LCM_REGISTER_CALLBACKS
