Description

I2S (Inter-IC Sound, Integrated Inter-chip Sound) is a serial bus interface that connects electronic sound devices together.
It is a bus standard developed by Philips for audio data transmission between digital audio equipment, using an independent wire
transmission clock and data signal design, by separating data and clock signals, to avoid distortion caused by time difference.


Feature

  1. Supports Philips I2S, Left-aligned Mode, Right-aligned Mode.
  2. Supports master mode or slave mode.
  3. Supports data length of 8/16/24/32 bits.
  4. Maximum I2S0 speed of 48 Mbps (master) or 32 Mbps (slave).
  5. Maximum I2S1/2 speed of 36 Mbps (master) or 32 Mbps (slave)
  6. The start phase of data sampling location selection is 1st-phase or 2nd-phase controlled register.
  7. Supports internally or externally controlled serial bit clock.
  8. Supports internally or externally controlled frame/sync.
  9. Programmable frame/sync.
  10. Programmable I2S format, including Zero padding, Left-aligned, Right-aligned.
  11. Programmable threshold interrupt of transmit/receive FIFO.
  12. Supports DMA transfer.
